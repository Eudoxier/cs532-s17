Chema Alonso desarrolla una nueva herramienta para evitar el ransomware

por Antonio Albarrán García · Publicado el 15 octubre, 2016

  En la actualidad el ransomware es la principal amenaza de seguridad para cualquier equipo informático. Este software se encarga básicamente de secuestrar tu ordenador, impidiéndote acceder a él, y pidiendo un rescate a cambio de quitar esta restricción. Y como algunos sabréis es bastante difícil evitar el Ransomware.

  Normalmente se transmite tanto como un troyano, como un gusano, infectando el sistema operativo, por ejemplo, con un archivo descargado o explotando una vulnerabilidad de software. En este punto, el ransomware se iniciará y cifrará los archivos del usuario con una determinada clave, que sólo el creador del ransomware conoce y proveerá al usuario que la reclame a cambio de un pago.

  Anteriormente, ya vimos una herramienta que te permitía evitar el ransomware, llamada Anti Ransom. Este enlace, te llevara a un artículo anterior, donde te explicamos como funciona Anti Ransom.

Latch ARW: la herramienta definitiva para evitar el ransomware

  Otra herramienta, que tal vez te llame mas la atención, es Latch ARW. Este nuevo software has sido desarrollado por el hacker y jefe de datos de Telefónica Chema Alonso. Latch ARW se encarga de proteger las carpetas cuyo contenido sea importante para el usuario de un ataque de ransomware.

  Una vez que las carpetas están protegidas con Latch, los archivos están disponibles para lectura pero no para borrado o escritura. Además, añade Alonso, “si un ransomware quisiera quitar la protección de Latch ARW, no podría hacerlo porque está protegido por el propio Latch, así que si el usuario no lo desbloquea el documento está a salvo”.

  También han realizado pruebas para comprobar su funcionamiento. Según dice Chema Alonso “Por supuesto, quisimos probarlo en un entorno real, así que le lanzamos una decena de muestras de ransomware para ver el comportamiento de Latch ARW con todos ellos y el resultado fue más que satisfactorio, ya que ninguno fue capaz de tocar ni uno solo de los documentos que estaban en las carpetas protegidas.”

  Latch ARW aun estará disponible en cuestión de días, y será totalmente gratuita. La aplicación, estará en fase beta, así que probablemente tendrá un par de fallos, aunque nada importante. De todas formas, tenéis toda la información en el blog oficial de Chema Alonso, donde encontrareis muchas cosas interesantes.

  Te recordamos que para este tipo de amenazas, siempre es importante tener un backup o copia de seguridad de todos los datos importantes que pueda haber en tu equipo, para no lamentar después pérdidas irreparables.
